Etiquetas HTML y sus atributos

Body y sus atributos:

El elemento "body" es parte de la estructura básica de un documento.

El elemento "body" es donde se pone todo el contenido de la página.

El elemento "body" tenía varios atributos en versiones antiguas de HTML.

Esta etiqueta solo tiene atributos generales.

P y sus atributos:

La etiqueta "p" se utiliza para escribir parágrafos en las páginas web.

Es importante acordarse de cerrar las etiquetas ya que sinó se quedan abiertas y no funcionan como deberían.

Esta etiqueta solo tiene atributos generales: id, style, class, dir y lang.

A y sus atributos:

El elemento "a" crea enlaces a diferentes páginas o secciones de la misma página.

Los vínculos son la parte más esencial de HTML.

Los vínculos pueden ser utilizados, para enlazar otros documentos

Sus atributos son: href y target.

IMG y sus atributos:

El elemento representa una imagen, es un recurso que se inserta en el documento.

Sus atributos son: src y alt.

DIV y sus atributos:

El elemento "div" es una etiqueta sin un significado semántico.

Se utiliza para dividir secciones y tener organizado el código.

Sus atributos son: main, header, footer, nav y article.
